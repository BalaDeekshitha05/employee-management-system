# Employee Attendance System - Frontend

## Setup

```sh
npm install
cp .env.example .env
npm start
```
- Update `REACT_APP_API_BASE` in `.env` for your backend.

## Features Scaffolded
- Register/Login (Zustand + Auth API)
- Protected routes (Employee/Manager)
- Dashboard stubs
- Material-UI theme

## Extend
- Add attendance, reports, calendar views, etc. to `/pages` and `/components`.
